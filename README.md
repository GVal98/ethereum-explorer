# Ethereum Explorer

Live demo: https://ethereum-explorer-gval98.vercel.app/

### Features:

- Blocks, Transactions, Balances Info
- Search
- Infinite scroll
- Dark theme

### Stack:

- Web3.js
- Next.js
- React
- React Query
- Mantine UI
- TypeScript

### Usage:

Install: `npm install` `npm run prepare`

Dev server: `npm run dev`

Prod Server: `npm run build` `npm start`
